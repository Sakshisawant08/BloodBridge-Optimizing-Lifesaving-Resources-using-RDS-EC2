BloodBridge: Optimizing Lifesaving Resources using RDS & EC2 is a smart blood donation and management system developed using Flask, MySQL, HTML, CSS, and JavaScript. The platform connects donors, hospitals, blood banks, and administrators to manage blood requests, donor records, and blood inventory efficiently.

The project is designed with AWS cloud architecture concepts such as Amazon EC2, RDS, Load Balancer, and CloudFront for scalable and secure deployment reference. BloodBridge aims to improve emergency blood availability and optimize lifesaving healthcare resources through a centralized digital platform.

Features
Donor Registration & Login
Hospital Blood Request Management
Blood Bank Inventory Tracking
Admin Dashboard
Emergency Blood Request System
Responsive UI
Cloud Architecture Reference using AWS Concepts
Technologies Used
Python Flask
MySQL
HTML, CSS, JavaScript
Bootstrap
AWS Concepts (EC2, RDS, VPC, CloudFront)
Objective

To create a centralized and efficient blood management platform that reduces emergency response time and improves blood resource accessibility.
